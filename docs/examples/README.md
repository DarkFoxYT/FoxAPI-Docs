# Examples Overview

The documents in this directory provide end-to-end walkthroughs that build on
the core FoxAPI helpers. Each guide focuses on a specific workflow and includes
copy-pasteable snippets you can adapt to your own mods.

* [`packets.md`](packets.md) – registering, sending and testing custom packets.
* [`nbt.md`](nbt.md) – practical NBT storage patterns for items, entities and block entities.
* [`content-registration.md`](content-registration.md) – complete builder usages for items, blocks, block entities and mobs.
* [`connected-textures.md`](connected-textures.md) – generating connected texture definitions with multi-loader compatibility helpers.
