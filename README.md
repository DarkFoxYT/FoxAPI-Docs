# FoxAPI Docs

This repository hosts shared tooling and helpers for Dark's modding
projects. Reference documentation lives under the `docs/` directory:

* [`docs/helpers.md`](docs/helpers.md) – one-stop catalog of every helper
  package shipped by FoxAPI.
* [`docs/coding-helper.md`](docs/coding-helper.md) – starter templates,
  checklists, and copy-pasteable stubs for new modders.
* [`docs/content-creation-helper.md`](docs/content-creation-helper.md) –
  overview of the fluent content registration builders.
* [`docs/data-driven-helper.md`](docs/data-driven-helper.md) – tips for
  emitting blockstates, models, and other JSON assets.
* [`docs/crafting-helper.md`](docs/crafting-helper.md) – JSON builders and
  serializer stubs for custom crafting grids.
* [`docs/networking-helper.md`](docs/networking-helper.md) – guidance for
  crafting and exercising custom packets with the networking sandbox.
* [`docs/nbt-helper.md`](docs/nbt-helper.md) – focused tips for working
  with the `NbtIO` utilities.
* [`docs/data-storage-helper.md`](docs/data-storage-helper.md) – server-side
  storage helpers for world, server, and per-player NBT.

Concrete walkthroughs with copy-pasteable snippets are collected in
[`docs/examples/`](docs/examples/).
